## 🔑 关键配置说明

### 1. 数据库连接(Supabase)

从Supabase获取的连接字符串格式:
```
postgresql://postgres:密码@db.xxxxx.supabase.co:5432/postgres
```

**注意**: 
- 密码中的特殊字符需要URL编码
- 例如: `p@ssw0rd` → `p%40ssw0rd`

### 2. JWT密钥生成

访问 https://randomkeygen.com 生成强密钥

选择 "CodeIgniter Encryption Keys" 那一行,复制任意一个。

### 3. CORS配置

确保后端允许前端域名访问:

```python
ALLOWED_ORIGINS = "https://*.streamlit.app,https://你的域名.com"
```

## ⚠️ 重要提醒

### 部署前必做

1. **备份代码**
   ```bash
   git add .
   git commit -m "部署前备份"
   git push
   ```

2. **检查敏感信息**
   - 确认 `.env` 文件未被提交
   - 确认 `secrets.toml` 未被提交
   - 检查 `.gitignore` 是否生效

3. **测试本地运行**
   ```bash
   # 后端
   cd backend
   python run_dev.py
   
   # 前端(新窗口)
   streamlit run app.py
   ```

### 部署后必做

1. **配置UptimeRobot**
   - 访问: https://uptimerobot.com
   - 每5分钟ping一次后端
   - 防止Render休眠

2. **功能测试**
   - 微信扫码登录
   - 文档上传转换
   - 余额扣减
   - 文件下载

3. **监控日志**
   - Render Dashboard查看后端日志
   - Streamlit Dashboard查看前端日志
   - Supabase查看数据库统计

### Q1: 部署需要编程基础吗?

### Q2: 多久可以上线?

### Q3: 真的完全免费吗?
**A**: 是的。三个平台都提供免费层,足够初期使用。

### Q4: 如果流量增长怎么办?
**A**: 
- Streamlit: 每月1000小时,足够10万+访问量

**A**:
