### 1. ✅ 对话框关闭按钮无效
**问题**: 所有弹出对话框的取消和关闭按钮点击后不能立刻关闭对话框，甚至是无效。

**根本原因**: 
- 对话框内的按钮使用了`st.rerun()`，导致页面重新加载而不是关闭对话框
- Streamlit的`@st.dialog`装饰器要求函数直接`return`才能关闭对话框

**修复方案**:
```python
# ❌ 错误的方式
if st.button("❌ 取消"):
    st.rerun()  # 重新加载页面，对话框不会关闭

# ✅ 正确的方式
if st.button("❌ 取消"):
    return  # 直接返回，对话框自动关闭
```

**已修复的文件**:
- `app.py` 第149行 - 反馈对话框提交按钮
- `app.py` 第156行 - 反馈对话框取消按钮
