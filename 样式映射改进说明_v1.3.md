# 样式映射功能改进说明 v1.3

## 更新日期
2026-04-24

## 改进概述

实现了更精细的文档样式映射管理功能，支持为每个文档独立配置样式映射规则。

---

## 核心改进

### 1. ✅ 三列表布局

**新增源文件列表**，样式信息区域现在包含三个列表：

```
┌─ 样式信息 ───────────────────────────────────────┐
│ [配置样式映射] (选择源文件后点击按钮配置...)     │
│                                                  │
│ ┌──────────┐  ┌──────────┐  ┌──────────┐      │
│ │源文件列表│  │源文档样式│  │模板样式  │      │
│ │(可点击)  │  │(自动更新)│  │(固定)    │      │
│ │• 文件A   │  │• Heading1│  │• Heading1│      │
│ │• 文件B   │  │• Normal  │  │• BodyText│      │
│ │• 文件C   │  │• List    │  │• List    │      │
│ └──────────┘  └──────────┘  └──────────┘      │
└──────────────────────────────────────────────────┘
```

**三个列表的作用：**

1. **源文件列表**（左侧）
   - 显示所有选择的源文件
   - 可点击选择单个文件
   - 单选模式

2. **源文档样式**（中间）
   - 显示选中文件的样式列表
   - 点击源文件时自动更新
   - 反映该文件实际使用的样式

3. **模板文档样式**（右侧）
   - 显示模板中可用的样式
   - 固定不变
   - 作为映射的目标选项

---

### 2. ✅ 联动机制

**工作流程：**

```
用户点击源文件列表中的某个文件
         ↓
触发 on_source_file_select() 事件
         ↓
从 file_style_maps 中获取该文件的样式
         ↓
清空源样式列表
         ↓
填充新的样式列表
         ↓
日志显示："已选择文件: xxx.docx (共X种样式)"
```

**代码实现：**

```python
def on_source_file_select(self, event=None):
    """源文件列表选择事件"""
    selection = self.file_listbox.curselection()
    if selection and len(self.source_files) > 0:
        index = selection[0]
        selected_file = self.source_files[index]
        self.selected_source_file = selected_file
        
        # 更新源样式列表
        if selected_file in self.file_style_maps:
            styles = self.file_style_maps[selected_file]['styles']
            self.source_listbox.delete(0, END)
            for style in sorted(styles):
                self.source_listbox.insert(END, style)
            
            self.log(f"已选择文件: {os.path.basename(selected_file)}")
```

### 4. ✅ 智能配置对话框

**打开逻辑：**

```python
def open_style_mapping(self):
    # 1. 确定要配置的文件
    if self.selected_source_file:
        # 有选中的文件，针对该文件配置
        target_file = self.selected_source_file
        target_styles = self.file_style_maps[target_file]['styles']
        current_mapping = self.file_style_maps[target_file].get('mapping', {})
    elif len(self.source_files) == 1:
        # 单文件模式
        target_file = self.source_files[0]
        ...
    else:
        messagebox.showwarning("请先在源文件列表中选择一个文件")
        return
    
    # 2. 打开对话框，只传入该文件的样式
    dialog = StyleMappingDialog(self.root, target_styles, self.template_styles, current_mapping)
    result = dialog.show()
    
    # 3. 保存映射到该文件
    if result is not None:
        self.file_style_maps[target_file]['mapping'] = result
```

**特点：**
- 只显示选中文件的样式
- 加载该文件已有的映射配置
- 保存时只更新该文件的配置
- 不影响其他文件的配置

### 5. ✅ 转换时使用对应配置

**单文件转换：**

```python
# 获取该文件的映射配置
file_mapping = None
if source in self.file_style_maps and self.file_style_maps[source].get('mapping'):
    file_mapping = self.file_style_maps[source]['mapping']
    self.log(f"使用该文件的自定义映射: {len(file_mapping)} 个样式")
elif self.custom_style_map:
    file_mapping = self.custom_style_map
    self.log(f"使用全局自定义映射")
else:
    self.log("使用系统默认映射规则")

# 执行转换
converter.full_convert(..., custom_style_map=file_mapping, ...)
```

**批量转换：**

```python
for source in self.source_files:
    # 每个文件使用各自的映射配置
    file_mapping = None
    if source in self.file_style_maps and self.file_style_maps[source].get('mapping'):
        file_mapping = self.file_style_maps[source]['mapping']
        self.log(f"使用该文件的自定义映射")
    else:
        self.log(f"使用系统默认映射")
    
    converter.full_convert(..., custom_style_map=file_mapping, ...)
```

### 3. 分析多个文件

```python
def analyze_multiple_source_styles(self, files):
    """分析多个源文件的样式（为每个文件单独存储）"""
    for file in files:
        try:
            doc = Document(file)
            styles = set()
            for para in doc.paragraphs:
                if para.style and para.style.name:
                    styles.add(para.style.name)
            
            # 保存该文件的样式
            self.file_style_maps[file] = {
                'styles': styles,
                'mapping': {}  # 初始为空，用户配置后填充
            }
        except Exception as e:
            self.log(f"分析文件失败 {os.path.basename(file)}: {e}")
```
