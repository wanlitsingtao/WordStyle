#### 模拟支付成功
```
POST /api/test-payment/simulate-payment
```
**请求**：
```json
{
  "order_id": "PAY20260501120000ABC123",
  "user_id": "abc123def456"
}
```

**响应**：
```json
{
  "success": true,
  "message": "✅ 支付成功！已充值 10,000 段",

  "paragraphs_added": 10000
```

- 模拟微信支付回调

### 3. 充值逻辑

```python
def recharge_user(user_id: str, amount: float, paragraphs: int) -> bool:
    """为用户充值"""
    # 1. 读取 user_data.json
    # 2. 增加余额和段落数
    all_data[user_id]['balance'] += amount
    all_data[user_id]['paragraphs_remaining'] += paragraphs
    
    # 3. 添加充值记录
    recharge_record = {
        'time': datetime.now().strftime('%Y-%m-%d %H:%M:%S'),
        'amount': amount,
        'paragraphs': paragraphs,
        'order_id': order_id,
        'method': 'wechat_scan'
    }
    all_data[user_id]['recharge_history'].append(recharge_record)
    
    # 4. 保存文件
    with open(USER_DATA_FILE, 'w', encoding='utf-8') as f:
        json.dump(all_data, f, ensure_ascii=False, indent=2)
```
