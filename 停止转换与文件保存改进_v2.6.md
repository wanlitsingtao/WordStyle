### 2. ✅ 改进文件被占用时的保存逻辑

**改进前**：
- 文件名后缀：`_HHMM`（时分）
- 示例：`document_1430.docx`
- 问题：同一分钟内多次保存会覆盖

**改进后**：
- 文件名后缀：`_YYYYMMDD_HHMMSS`（年月日_时分秒）
- 示例：`document_20260425_143025.docx`
- 优势：确保唯一性，不会覆盖

**触发条件**：
- 原始输出文件被其他程序占用
- Word 正在打开该文件
- 文件资源管理器选中了该文件

#### 3. 停止转换方法

```python
def stop_conversion(self):
    """停止转换"""
    if self.conversion_thread and self.conversion_thread.is_alive():
        self.stop_conversion_flag = True
        self.log("\n⚠ 正在停止转换，请稍候...")
        self.stop_btn.config(state='disabled')
        self.convert_btn.config(state='disabled')
```

#### 4. 启动转换时重置标志

```python
def start_conversion(self):
    # 重置停止标志
    self.stop_conversion_flag = False
    
    # 保存线程引用
    self.conversion_thread = threading.Thread(target=self.do_conversion)
    self.conversion_thread.daemon = True
    self.conversion_thread.start()
    
    # 启用停止按钮，禁用开始按钮
    self.stop_btn.config(state='normal')
    self.convert_btn.config(state='disabled')
```

#### 5. 检查停止标志

```python
# 单文件模式
if len(self.source_files) == 1:
    if self.stop_conversion_flag:
        self.log("\n⚠ 转换已停止")
        return

# 多文件模式
for idx, source in enumerate(self.source_files, 1):
    if self.stop_conversion_flag:
        self.log("\n⚠ 转换已停止")
        break
```

#### 6. 恢复按钮状态

```python
finally:
    # 恢复按钮状态
    def restore_buttons():
        self.stop_btn.config(state='disabled')
        self.convert_btn.config(state='normal')
    self.root.after(0, restore_buttons)
```

#### 7. 改进的文件保存

```python
def save_with_retry(self, doc, output_file, max_retries=10):
    current_file = output_file
    for attempt in range(max_retries):
        try:
            doc.save(current_file)
            return True, current_file, f"文档已保存到 {current_file}"
        except (PermissionError, OSError, IOError) as e:
            # 生成新的文件名：基础名 + _YYYYMMDD_HHMMSS + 扩展名
            base, ext = os.path.splitext(output_file)
            time_suffix = datetime.now().strftime("_%Y%m%d_%H%M%S")
            current_file = f"{base}{time_suffix}{ext}"
            print(f"  尝试备用文件名: {current_file}")
            time.sleep(0.5)
```

## 💡 使用建议

### 停止转换功能

**何时使用**：
1. 发现选错了源文件或模板
2. 转换时间过长，不想继续等待
3. 需要立即释放系统资源
4. 发现了配置错误需要修正

**注意事项**：
- ⚠️ 停止不是即时的，需要等待当前文件处理完成
- ⚠️ 已完成的文件会保留，未完成的可能不完整
- ⚠️ 停止后需要手动清理可能的临时文件

**操作流程**：
```
1. 点击"开始转换"
2. 转换进行中...
3. 发现问题，点击"停止转换"
4. 等待当前文件处理完成
5. 看到"转换已停止"提示
6. 按钮恢复，可以重新配置
```

### 文件保存改进

**触发场景**：
1. Word 正在打开输出文件
2. 文件资源管理器选中了输出文件
3. 其他程序锁定了输出文件

**文件名示例**：
```
原始文件名: document_converted.docx

第一次失败: document_converted_20260425_143025.docx
第二次失败: document_converted_20260425_143026.docx
第三次失败: document_converted_20260425_143027.docx
```

**优点**：
- ✅ 包含完整日期，便于识别
- ✅ 精确到秒，避免冲突
- ✅ 按时间排序，易于管理
- ✅ 不会覆盖已有文件

**管理建议**：
- 定期清理带时间戳的备份文件
- 保留最新的文件，删除旧的
- 可以使用搜索功能查找特定日期的文件
