## 完整操作流程演示

### 场景1：单文件转换（带自定义样式映射）

**步骤1：选择文件**
```
用户点击"浏览..." → 选择"技术方案.docx"
用户点击"浏览..." → 选择"标准模板.docx"
```

**步骤2：查看样式信息**
```
左侧列表显示源文档样式：
  • Heading 1
  • Heading 2
  • Normal
  • List Paragraph
  • Quote

右侧列表显示模板文档样式：
  • Heading 1
  • Heading 2
  • Normal
  • Body Text
  • List Bullet
```

**步骤3：配置自定义映射**
```
用户勾选"使用自定义样式映射"
弹出对话框：

源文档样式          →    模板样式
─────────────────────────────────────
Heading 1          →    Heading 1      ✓
Heading 2          →    Heading 2      ✓
Normal             →    Body Text      ← 用户修改
List Paragraph     →    List Bullet    ← 用户修改
Quote              →    Normal         ← 用户修改

用户点击"确定"
```

**步骤4：配置其他选项**
```
☑ 进行祈使语气转换
应答句文本: [应答：本投标人理解并满足要求。]
应答句样式: [应答句]
```

**步骤5：开始转换**
```
用户点击"开始转换"

日志区域显示：

## 转换效果对比

### 转换前（源文档）
```
第一章 项目概述                    [样式: Heading 1 - 蓝色, 16pt]
本项目必须在规定时间内完成。        [样式: Normal - 黑色, 12pt]
投标人应当具备相关资质。            [样式: Normal - 黑色, 12pt]

1.1 项目背景                      [样式: Heading 2 - 蓝色, 14pt]
应充分理解项目需求。不得忽视细节。  [样式: Normal - 黑色, 12pt]
```

### 转换后（输出文档）
```
第一章 项目概述                    [样式: Heading 1 - 模板样式]
应答：本投标人理解并满足要求。      [样式: 应答句 - 新增]
本项目将在规定时间内完成。          [样式: Body Text - 模板样式]
本投标人应当具备相关资质。          [样式: Body Text - 模板样式]

1.1 项目背景                      [样式: Heading 2 - 模板样式]
应答：本投标人理解并满足要求。      [样式: 应答句 - 新增]
将充分理解项目需求。不会忽视细节。  [样式: Body Text - 模板样式]
```

**变化说明：**
1. ✓ 标题样式转换为模板样式
2. ✓ 每个标题后插入了应答句
3. ✓ "必须" → "将"
4. ✓ "应" → "将"
5. ✓ "不得" → "不会"
6. ✓ "投标人" → "本投标人"
7. ✓ 正文样式从"Normal"改为"Body Text"（模板样式）

## 高级功能演示

### 自定义应答句

```
用户修改配置：
应答句文本: [注：此部分内容由AI辅助生成，请人工审核。]
应答句样式: [注释]

转换后效果：
第一章 项目概述
注：此部分内容由AI辅助生成，请人工审核。  [样式: 注释]
```

### 跳过语气转换

```
用户取消勾选"进行祈使语气转换"

转换结果：
- 样式仍然会转换
- 应答句仍会插入
- 但"必须"、"应"等词汇保持不变
```

## 错误处理演示

### 场景：模板文件不存在
```
用户点击"开始转换"

弹出错误框：
┌────────────────────────────┐
│         警告               │
├────────────────────────────┤
│ 请选择模板文档             │

```
日志显示：
[10:40:15] 警告：样式缺失: 源样式 'CustomStyle' 无映射且模板中不存在，使用 'Normal'

```

自动使用备用文件名保存

## 性能参考

| 文档大小 | 段落数 | 转换时间 | 说明 |
|---------|--------|----------|------|
| 小文档 | < 50 | ~2秒 | 包含样式+语气+应答句 |
| 中文档 | 50-200 | ~5秒 | 包含样式+语气+应答句 |
| 大文档 | 200-500 | ~15秒 | 包含样式+语气+应答句 |
| 超大文档 | > 500 | ~30秒+ | 建议分批处理 |

*测试环境：Windows 10, Python 3.14, Intel i5处理器*
