```python
def clean_list_numbering(text):
    """清理开头数字编号：1、 1） (1) 等"""
    pattern = r'^\s*(?:\d+[、\)）]|[（(]\d+[）\)])\s*'
    cleaned = re.sub(pattern, '', text, count=1)
    return cleaned
```

**功能**：
- 自动清理列表项开头的数字编号

## 💡 使用建议

### 自定义列表符号

**GUI 中配置列表符号**：

1. **启动 GUI**
2. **找到“样式信息”区域**
3. **看到“列表段落符号:”输入框**
4. **输入您喜欢的符号**，例如：
   - `● `（默认，黑色圆点）
   - `- `（减号）
   - `* `（星号）
   - `→ `（箭头）
   - `■ `（方块）
   - `◆ `（菱形）
5. **点击“开始转换”**

**注意**：
- 符号后建议加一个空格，使排版更美观
- 可以输入多个字符，如 `>> `、`-> ` 等
- 修改后的设置仅对当前转换生效

### 对于包含列表的文档

新版本的列表处理更加完善：

1. **自动编号清理**
   - 如果源文档列表有编号（如 `1、第一项`）
   - 转换后会变成 `● 第一项`
   - 编号被自动移除

2. **统一的列表符号**
   - 所有列表都使用 `● ` 符号
   - 视觉效果更统一
   - 符合大多数文档规范

### 对于包含 Visio 图的文档

新的提示位置让用户更早注意到：

1. **启动 GUI**
2. **立即看到橙色提示**
3. **想起需要转换 Visio 图**
4. **再进行转换操作**
