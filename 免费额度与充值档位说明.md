# 免费额度与充值档位说明

## 📝 更新日期
2026-05-02

---

## 💡 重要说明

### 1. 免费额度（首次访问赠送）
**默认值**: **10,000 段** ✅

**获取方式**: 
- 新用户首次访问网站时自动获得
- 无需充值，无需登录
- 每个用户只能领取一次

**配置位置**:
- 前端: `app.py` - `get_free_paragraphs_config()` 函数
- 后端: `backend/app/api/admin.py` - 数据库配置
- 管理后台: `admin_dashboard.py` - 系统配置页面

**代码示例**:
```python
# app.py 第161-171行
def get_free_paragraphs_config():
    """从后端获取免费额度配置"""
    try:
        import requests
        response = requests.get(f"{BACKEND_URL}/api/admin/config/free-paragraphs", timeout=2)
        if response.status_code == 200:
            data = response.json()
            return data.get('config_value', 10000)  # 默认10000
        return 10000
    except:
        return 10000  # 默认值
```

### 2. 充值档位（付费购买）

#### 体验版
**价格**: ¥1 元  
**段落数**: **1,000 段** ✅

**说明**: 
- 这是**付费充值**的最低档位
- 与免费额度（10000段）是**不同的概念**
- 适合小额测试使用

**配置位置**: `app.py` 第31-35行


## ❓ 常见疑问

### Q1: 为什么免费额度是10000，而充值1元只有1000？

**A**: 这是有意设计的：
- **免费额度（10000段）**: 作为"试用装"，让用户充分体验功能
- **充值档位（1元=1000段）**: 作为"正式服务"，按量付费

**性价比对比**:
- 免费额度: 10000段 / ¥0 = ∞ (无限性价比)
- 体验版: 1000段 / ¥1 = 1000段/元
- 标准版: 6000段 / ¥5 = 1200段/元
- 专业版: 13000段 / ¥10 = 1300段/元
- 企业版: 70000段 / ¥50 = 1400段/元

**结论**: 充值越多，性价比越高！

### Q2: 我在哪里看到了1000而不是10000？

可能的情况：

#### 情况1: 侧边栏显示余额为1000
**原因**: 该用户充值了1元（体验版），获得了1000段

**验证方法**:
```json
// user_data.json 中查看
{
  "user_xxx": {
    "paragraphs_remaining": 1000,  // 这是充值后的余额
    "recharge_history": [
      {
        "amount": 1,
        "paragraphs": 1000,  // 充值1元获得1000段
        "package": "体验版"
      }
    ]
  }
}
```

---

#### 情况2: Toast提示显示"已赠送您 1000 段"
**原因**: 可能是后端API返回的配置值是1000

**检查方法**:
1. 访问后端API: http://localhost:8000/api/admin/config/free-paragraphs
2. 查看返回的 `config_value` 字段
3. 如果不是10000，需要在管理后台修改

**修复方法**:
```bash
# 方法1: 通过管理后台修改
访问 http://localhost:8503 -> 系统配置 -> 修改为10000 -> 保存

# 方法2: 直接修改数据库（需要SQL知识）
UPDATE system_configs SET config_value = '10000' 
WHERE config_key = 'free_paragraphs_on_first_login';
```

---

#### 情况3: 新用户首次访问只获得1000段
**原因**: 后端配置被修改过

**验证步骤**:
1. 打开浏览器无痕窗口
2. 访问 http://localhost:8501
3. 查看Toast提示和侧边栏余额
4. 如果显示1000，说明配置有问题

**修复方法**: 同上（通过管理后台或数据库修改）

## 🔧 如何修改免费额度

### 方法1: 通过管理后台（推荐）

1. 访问管理后台: http://localhost:8503
2. 点击左侧菜单 "⚙️ 系统配置"
3. 找到 "🎁 免费额度设置"
4. 修改 "新用户首次登录赠送段落数" 为 **10000**
5. 点击 "💾 保存配置"

---

### 方法2: 通过后端API

```bash
# 查看当前配置
curl http://localhost:8000/api/admin/config/free-paragraphs

# 修改配置
curl -X PUT http://localhost:8000/api/admin/config/free-paragraphs \
  -H "Content-Type: application/json" \
  -d '{"free_paragraphs": 10000}'
```

---

### 方法3: 直接修改数据库

```sql
-- 连接数据库
sqlite3 backend/app/data.db

-- 查看当前配置
SELECT * FROM system_configs WHERE config_key = 'free_paragraphs_on_first_login';

-- 修改配置
UPDATE system_configs 
SET config_value = '10000' 
WHERE config_key = 'free_paragraphs_on_first_login';

-- 验证修改
SELECT * FROM system_configs WHERE config_key = 'free_paragraphs_on_first_login';
```
