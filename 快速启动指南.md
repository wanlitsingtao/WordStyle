## 🔄 日常启动

### 方式一：一键启动（推荐）⭐

双击运行：**[启动完整版.bat](file://e:\LingMa\WordStyle\启动完整版.bat)**

这个脚本会：
1. ✅ 启动后端服务（新窗口）
2. ✅ 等待5秒
3. ✅ 启动前端服务（新窗口）
4. ✅ 自动打开浏览器

### 方式二：手动启动

```bash
# 终端1：启动后端
cd backend
venv\Scripts\python.exe run_dev.py

# 终端2：启动前端
cd e:\LingMa\WordStyle
.venv\Scripts\python.exe -m streamlit run app_with_wechat_login.py --server.port=8502
```

## 🔍 故障排查

### 问题1：后端服务未启动

**症状**：前端显示 "❌ 后端服务未启动"

**解决**：
```bash
cd backend
venv\Scripts\python.exe run_dev.py
```

**验证**：访问 http://localhost:8000/health，应返回 `{"status":"healthy"}`

---

### 问题2：端口被占用

**症状**：启动时提示 "Address already in use"

**解决**：
```bash
# 查找占用端口的进程
netstat -ano | findstr :8000  # 后端
netstat -ano | findstr :8502  # 前端

# 终止进程
taskkill /F /PID <进程ID>
```

或者修改端口：
```bash
# 修改前端端口
.venv\Scripts\python.exe -m streamlit run app_with_wechat_login.py --server.port=8503
```

---

### 问题3：数据库不存在

**症状**：后端启动时报错

**解决**：
```bash
cd backend
重新初始化数据库.bat
```

---

### 问题4：虚拟环境问题

**症状**：提示 "No module named xxx"

**解决**：
```bash
cd backend
修复环境.bat
```
